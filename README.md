# Tinta Agent

Home Assistant App для платформы Tinta Lab.

## Установка

[![Add to Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Ftinta-lab%2Ftinta-agent)

После добавления репозитория найди **Tinta Agent** в магазине Apps → Install.
